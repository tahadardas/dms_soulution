# DMS SOULUTION — دليل تثبيت البرنامج على Windows

هذا الدليل مخصص للمستخدم النهائي (صاحب المطعم أو الكاشير) لتثبيت النظام على جهاز جديد.

## 1. المتطلبات
- نظام تشغيل Windows 10 أو أحدث (64-bit).
- مساحة فارغة لا تقل عن 500 ميجابايت.
- لا يحتاج البرنامج لوجود Node.js أو أي برامج إضافية؛ كل شيء مضمن في الملف.

## 2. خطوات التثبيت
1. قم بتحميل ملف التثبيت: `DMS-SOULUTION-Setup.exe`.
2. اضغط مرتين على الملف لبدء التثبيت.
3. اتبع معالج التثبيت (Setup Wizard).
4. بعد الانتهاء، ستظهر أيقونة البرنامج "DMS SOULUTION" على سطح المكتب.

## 3. التشغيل الأول (Setup Wizard)
عند فتح البرنامج لأول مرة، ستظهر لك شاشة الإعدادات الأولية:
- **اسم المطعم**: أدخل الاسم الذي ترغب بظهوره في الفواتير.
- **إعدادات المدير**: قم بتعيين اسم مستخدم وكلمة مرور للمدير (أول مستخدم).
- **إعدادات الطابعة**: قم باختيار الطابعات المتصلة بالجهاز (طابعة الكاشير / طابعة المطبخ).

## 4. مسارات البيانات (مهم جداً)
يقوم البرنامج بتخزين بياناتك في مجلدات ثابتة لضمان عدم ضياعها عند التحديث:
- **قاعدة البيانات**: `C:\DMS\data\dms.db`
- **النسخ الاحتياطي**: `C:\DMS\backups\`
- **السجلات**: `C:\DMS\logs\`

## 5. النسخ الاحتياطي (Backup)
ينصح البرنامج دائماً بعمل نسخة احتياطية يومية. يمكنك القيام بذلك يدوياً من خلال الإعدادات داخل البرنامج، أو سيقوم النظام بعمل نسخة تلقائية عند كل تحديث.

## 6. حل المشاكل الشائعة
- **البرنامج لا يفتح**: تأكد من أن برنامج الحماية (Antivirus) لا يمنع API من التشغيل.
- **الطابعة لا تعمل**: تأكد من تعريف الطابعة على Windows أولاً، ثم اخترها من إعدادات البرنامج.
- **تعذر الاتصال بالخدمة**: قم بإغلاق البرنامج وفتحه مجدداً؛ سيقوم البرنامج تلقائياً بمحاولة تشغيل الخدمة الخلفية.

للدعم الفني، يرجى مراجعة سجلات النظام في مجلد `C:\DMS\logs\`.
