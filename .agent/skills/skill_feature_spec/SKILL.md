---
name: Feature Spec Writer
description: Produces documentation-first feature specs aligned with problem-first/data-first rules.
---

## Steps
1) Problem Statement + Users/Roles
2) Success Metrics + Acceptance Criteria
3) Data Model impact (entities/constraints)
4) Business Rules (including accounting/cost rules if relevant)
5) API Contracts (request/response/errors/examples)
6) UI Flow + States (loading/error/empty)
7) Test Plan (unit/integration/e2e)
